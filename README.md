# 🌟 2Pac Tribute Website

Este é um projeto pessoal desenvolvido como um tributo ao artista **Tupac Shakur**, utilizando **HTML** , **CSS** e **JS**, com foco em conteúdo visual, estrutura e organização de informações biográficas e musicais.

## 🎯 Objetivo

Criar um site visualmente atrativo para treinar habilidades em:

- Estruturação com HTML
- Estilização com CSS puro
- Interatividade com JavaScript
- Layout responsivo (em progresso)
- Organização de conteúdo temático

## 📱 Responsividade

Responsividade para dispositivos móveis e tablets está sendo implementada aos poucos.  
Atualmente, o site funciona melhor em telas de desktop e notebooks.

## 🛠️ Tecnologias

- HTML5
- CSS3 (incluindo Media Queries)
- JavaScript
- Vercel (para deploy)

## 🧠 Lições aprendidas

- Organização de seções e semântica em HTML
- Uso de `addEventListener` para criar interatividade
- Manipulação do DOM com JavaScript puro
- Uso de Flexbox e grid layout
- Implementação de media queries
- Adaptação visual para telas menores
